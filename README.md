# 📄 Django File Converter (PDF ↔ DOCX)

A minimal and clean Django web application to convert files between **PDF and DOCX formats**.

---

## 🚀 Features

- 📂 Upload `.pdf` or `.docx` files
- 🔄 Convert:
  - PDF → DOCX
  - DOCX → PDF
- ⬇️ Instant download after conversion
- ❌ File type validation
- 💡 No database required
- 🌍 Cross-platform (uses LibreOffice)

---

## 🛠️ Tech Stack

- Python
- Django
- pdf2docx
- LibreOffice (for DOCX → PDF)
